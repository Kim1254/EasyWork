> ## EasyResume
사회적 약자를 위한 이력 관리 시스템

### 프론트엔드(React) 구성 환경
- npm: 9.5.1
- React: 18.2.0

### 백엔드(Django) 구성 환경
- Python: 3.9.0
- Django: 4.1.2
- Requirements: 아래 코드를 실행하세요.

```bash
cd ${path_to_EasyResume}
python -m pip install requirements.txt
```

### 사용 오픈소스 목록
- [OpenAI/Whisper](https://github.com/openai/whisper)
